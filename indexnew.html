QUESTiFY- Turn Learning into a Quest

QUESTiFY is a powerful tool designed to help students and educators create customizable quizzes for aptitude and technical preparation. With AI-powered question generation and performance tracking, it provides a seamless learning experience.

How It Works

1

Input Text

Paste your text or upload a document. The system processes the text to extract key concepts and sentences.

2

Summarize Text

The text is summarized using a T5 Transformer model to focus on the most important information.

3

Extract Keywords

Key phrases and nouns are extracted using MultipartiteRank and Spacy for question generation.

4

Generate Questions

Questions are generated using a T5 SQuAD model, which creates contextually accurate MCQs.

5

Create Distractors

Distractors (incorrect options) are generated using Sense2Vec and WordNet for plausible alternatives.

6

Display MCQs

The final MCQs are displayed with options, correct answers, and explanations for effective learning.
